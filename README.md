# Zeetachec RF Architecture

This repository contains the RF communication architecture, test firmware, and protocol libraries for the Zeetachec Maritime Sensor System.

## Current focus
- Validate RF architecture over LoRa
- Support buoy-side simulated fuze traffic
- Support controller-side command and ACK validation
- Prepare for future ESP32 + WiFi bridge

## Main components
- ZeetaRfLink
- ZeetaAcomms16
- Buoy_RF_LoRa_Test
- RF_Controller_LoRa_Test
